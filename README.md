# QuickToolsHub

A multi-tool platform built with Next.js + Tailwind CSS. Each tool has an SEO-friendly route, for example:

- `/random-name-picker`
- `/password-generator`
- `/qr-code-generator`
- `/random-number-generator`
- `/unit-converter`
- `/image-compressor`
- `/json-formatter`
- `/text-tools`
- `/color-picker`
- `/spin-wheel`

## Included tools

- Random Name Picker
- Password Generator
- QR Code Generator
- Random Number Generator
- Unit Converter
- Image Compressor
- JSON Formatter
- Text Tools (case conversion + word count)
- Color Picker
- Spin Wheel / Decision Maker

## Tech stack

- Next.js (App Router) + React + TypeScript
- Tailwind CSS
- Google Analytics (GA4) integration via env var
- Google AdSense integration with reusable ad slots
- Ready for Vercel deployment + GitHub auto-deploy

## Local development

1. Install dependencies:

	```bash
	npm install
	```

2. Create env file:

	```bash
	cp .env.example .env.local
	```

3. Fill your IDs in `.env.local`:

	- `NEXT_PUBLIC_GA_ID`
	- `NEXT_PUBLIC_ADSENSE_CLIENT`

4. Start development server:

	```bash
	npm run dev
	```

5. Open `http://localhost:3000`.

## Production build

```bash
npm run build
npm run start
```

## Deploy on Vercel

1. Push this repo to GitHub.
2. Import the repo in Vercel.
3. Add the same env vars in Vercel Project Settings.
4. Connect your custom domain (e.g. `quicktoolshub.com`, `smarttoolbox.io`, `toolnest.io`).
5. Enable automatic deployments from your main branch.

## Notes

- SEO files are included: `sitemap.xml` and `robots.txt` (generated by Next.js metadata routes).
- Tools run client-side for speed and minimal backend complexity.
- Ad slots are placeholders until a valid AdSense client and slot IDs are configured.
